# GCE-CP2-Differential-Equations

Powerpoint used for teaching Chapter 7: Differential Equations from Pearson Edexcel GCE A-Level Further Mathematics Core Pure Mathematics Book 2. 

Slideshow made using [Advanced Slides](https://github.com/MSzturc/obsidian-advanced-slides), an Obsidian Plugin using reveal.js. 

Alongside with the main slides, there are also some proof sheets that are "insignificant" for the curriculum.
